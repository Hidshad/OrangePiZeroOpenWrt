# OrangePiZeroOpenWrt
2020/01/02


香橙派Zero的OpenWrt


由lede那么编译而成，在默认的配置下添加了两套皮肤以及ssl，支持四种飞机模式



![](https://github.com/Hidshad/OrangePiZeroOpenWrt/blob/master/2020-01-12_014657.png)



默认网关：192.168.1.1 

默认账号：root

默认密码：password

！！！注意此款镜像为香橙派zero，非zero plus。他们的处理器是不一样的，不相兼容。

如果想自己编译的话可以去原作者地址：

https://github.com/coolsnowwolf/lede
